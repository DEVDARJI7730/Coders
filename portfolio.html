Netflix Clone

Netflix Clone is a front-end web project built using HTML, CSS, and JavaScript that replicates the look and feel of the popular streaming platform Netflix. It features a responsive user interface with a hero banner, movie thumbnails, hover effects, navigation bar, and category-based sections. It's a great project to demonstrate front-end development skills and UI/UX creativity.

Online Shoes

Online Shoes Store is a responsive and interactive website created using HTML, CSS, and JavaScript, designed to showcase and sell footwear products. The site features a modern homepage with a navigation bar, product listings, filter options, and an animated product carousel. Each shoe item includes images, prices, and an "Add to Cart" button with JavaScript-driven interactivity. CSS animations and transitions enhance the user experience, while a responsive layout ensures the website looks great on all devices. This project highlights core front-end development skills and e-commerce website design.

Speed Typing

Speed Typing Game is a fun and interactive web application built using HTML, CSS, and JavaScript that tests how fast and accurately users can type. The interface displays a random sentence that the user must type as quickly as possible, with real-time feedback, timers, and error highlighting. JavaScript handles the logic for word matching, timer countdown, and accuracy calculation, while CSS ensures a clean, responsive design with engaging visuals. This project is perfect for practicing DOM manipulation, event handling, and creating an engaging user experience with pure front-end technologies.
